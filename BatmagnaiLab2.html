How to define a table?
A table is defined in HTML using the <table>element. It's used to display data in a grid format with rows and columns, similar to a spreadsheet.
When to use the following HTML elements:
  • <tr>: Table Row. Used to define a single horizontal row of cells in a table.
  • <th>: Table Header. Used for table header cells; text is typically rendered as bold and centered.
  • <td>: Table Data. Used for standard data cells within a table row.
  • <thead>: Table Head. A container for all the header rows (<tr> elements with <th> cells) at the top of the table.
  • <tfoot>: Table Foot. A container for content (like summaries or totals) that should logically appear at the bottom of the table.
  • <caption>: Table Caption. Provides a descriptive title or explanation for the table, usually displayed above it.
What is the difference between colspan and rowspan?
These are attributes used inside <th> or <td> tags to merge cells:
  • colspan: Specifies how many columns a cell should span (merge horizontally).
  • rowspan: Specifies how many rows a single cell should span (merge vertically).
What is the difference between cell spacing and padding?
  • Cell Padding (CSS padding): The space inside a cell, between the content and the cell's border.
  • Cell Spacing (HTML cellspacing or CSS border-spacing): The space between the individual cells (borders) of a table.
How do you define table borders using CSS?
You define the border on the table element and its cells, typically using the border property: table, th, td { border: 1px solid black; }. You can also use border-collapse: collapse; to remove the space between cell borders.
How do you control border widths using CSS?
The border width is controlled by the first value in the border shorthand property (e.g., the 1px in border: 1px solid black;) or by using the specific property border-width: border-width: 2px;
